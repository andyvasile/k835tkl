# k835tkl
Distribución en español para el teclado Logitech k835 TKL para Mac. Hecho con Ukelele.
https://www.logitech.com/es-ar/products/keyboards/k835-tkl-mechanical.920-010085.html?crid=27

El teclado tiene exactamente la misma distribución física que el Magic Keyboard, pero MacOSx mapea las teclas de forma distinta (principalmente "<" y "º"). Usando Ukelele (https://software.sil.org/ukelele/) logré remapear las teclas para que funcionen correctamente en Mac - MacOS X 11.3.

Descargar, añadir al sistema con doble click, y seleccionar desde Preferencias de Sistema > Teclado > Fuentes de entrada.
